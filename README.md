# PyTradingSystem
计划写一个纯Python语言编写的股票期货交易系统，初衷是技术交流，一定是轻量级的，但是也能够做实盘交易，先立个Flag，希望有对此感兴趣的朋友一起来完成。

这个系统的订单模块应该不存在平今平昨的概念，不存在开平仓概念，也不存在交易日的概念，只有多空、买卖，并且可以7x24连续运行，不需要结算，平今平昨、开平仓、交易日等概念只存在于某些报盘中，如CTP报盘。
